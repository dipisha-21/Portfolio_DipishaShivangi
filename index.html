Most recent work

Automatic Number Plate Detection Model

-This project aims to develop an effective ANPR system using EasyOCR for text recognition and the YOLOv8 model for object identification. The research uses a large dataset of annotated car photos, focusing on preprocessing methods like augmentation and scaling. The YOLOv8 model is trained to recognize license plates with high precision, and EasyOCR extracts alphanumeric characters. The system has an overall accuracy of 99.5%.
